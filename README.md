🔐 Gerador de Senhas Seguras








Uma aplicação web que permite gerar senhas seguras e aleatórias de forma rápida e simples.
Ideal para quem precisa criar senhas fortes para proteger contas e sistemas.

🚀 Funcionalidades

🔑 Geração de senhas aleatórias

🔢 Escolha do tamanho da senha

🔣 Inclusão de letras, números e símbolos

📋 Botão para copiar a senha gerada

🎨 Interface simples e intuitiva
